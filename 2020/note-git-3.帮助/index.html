[Note] Git-3.帮助

Written by Wang Kefeng with ♥ on August 23, 2020 in Note

Git 帮助

  • 查看完整手册
  • 查看简短的可用选项快速参考
  • 联网获取帮助

查看完整手册

以下三种方式是等效的：

$ git help <verb>
$ git <verb> --help
$ man git-<verb>

e.g.:

$ git help config
$ git config --help
$ man git-config

查看简短的可用选项快速参考

使用如下命令查看 Git 的某个操作的可用参数、使用方法和简单描述：

$ git <verb> -h

联网获取帮助

通过搜索引擎总是能获取到一些有用的资料，IRC 的 #git 和 github 都是很好的可以寻求帮助的频道。当然还有各个 QQ 群等。
